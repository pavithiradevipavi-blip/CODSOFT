# Movie Rating Prediction

This project predicts IMDb movie ratings using machine learning techniques.

## Dataset
IMDb India Movies Dataset from Kaggle  
[Link to Dataset](https://www.kaggle.com/datasets/adrianmcmahon/imdb-india-movies)

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Google Colab

## Model
- Linear Regression

## Workflow
1. Data cleaning and preprocessing
2. Feature selection: Year of release, Number of votes
3. Train-test split (80% / 20%)
4. Model training and testing

## Results
- The model predicts movie ratings based on year and votes.
- Example prediction:  
  For a movie released in 2022 with 100,000 votes → Predicted rating ≈ 7.1
